## 创建人物的相册列表

对每一个人物, 在数据库中保存以下条目:

```yml
    id: '681442'
    name: '白岩松'
    num_album: 11
    num_photo: 82

    lemma_id: '36809'
    sub_lemma_id: '36809'
    new_lemma_id: '681442'

    // For each album:
        album_id: '126741'
        album_title: '精彩图册'
        album_num_photo: 4
    ...
```
